Research

Enterprise Mobile Computing Systems, Algorithms and Applications

Taskr

Codeless Application Mobilization

Taskr is a fast and easy application mobilization solution that allows users to mobilize simple workflows without writing code and perform them using a custom-tailored mobile app, through twitter, text or email.

Trackr and DashBoard

Front-end APIfication of Web Applications

Trackr is a robust and easy javascript framework that reliably tracks any UI element within a web application even as it's layout changes. Dashboard is an Android application that uses Trackr framework to monitor values across many web applications through a single application.

Precog

Time-shifted WiFi Offloading

Precog predicts the web content content a user is likely to access on cellular in the future and prefetches it while the user is connected to WiFi.

Dejavu

Email Response Assistant

Dejavu is an assitant integrated within K-9 email client, which provides suggestion nuggets related to an Inbox email from the information contained within the user's past emails.

Peek

Mobile-to-Mobile Remote Computing

Peek is a remote computing protocol, specifically designed for smartphones and tablets, that allows users to log-in to another smartphone/tablet and operate it locally.

E-Nudge

Behavioral Nudges for Email

This is an intelligent email assistant that reorders user's emails for improved productivity based on behavioral insights from various psychology studies.

Pulsar

LTE Throughput Estimation

Pulsar is a metric that accurately estimates the throughput of users within an enterprise LTE small cell.
